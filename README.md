# Slither Arena

A multiplayer snake game with custom face upload functionality.

## Features

- Real-time multiplayer gameplay
- Custom face upload for snake heads
- AI opponents
- Leaderboard system
- Player progression and XP
- Multiple skins and colors
- Mobile-friendly controls

## Custom Face Upload

Players can upload a custom face image that appears on their snake's head:

- Upload from file picker or camera
- Built-in image cropper with zoom, rotate, and positioning
- Automatic image processing and optimization
- Rate limited to 3 uploads per 24 hours
- Maximum file size: 5MB
- Supported formats: JPG, PNG, WebP

## Getting Started

1. Enter your username
2. Choose your snake color
3. Select a skin
4. Optionally upload a custom face for your snake head
5. Click "Start Game" to play

## Controls

- **Desktop**: Move with mouse, boost with Space or Left Click
- **Mobile**: Use on-screen joystick and boost button
